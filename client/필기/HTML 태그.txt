
<!-- <p>, paragraph -->
    - 문단 태그
    - 문단을 표현(=텍스트 집합)
    - 쌍태그(<p>문단의내용</p>)
    - 혼합용(텍스트 + 자식 태그)
    
    - 속성
        ~ align : 내용물의 수평 정렬
            - horizontal alignment
            - 열거형 > left|center|right|justify(양쪽정렬)
    

<!-- <hn>, heading -->
    - 제목 태그
    - n: 1~6
    - <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
    - 쌍태그
    - 혼합형
    - 숫자는 중요도를 나타낸다. > 보이는 형태는 중요하지 않음!
        ~ <h1>: 문서 내에서 가장 중요한 제목
        ~ <h6>: 문서 내에서 가장 덜 중요한 제목

    - 속성
        ~ align: 수평 정렬

<!-- <hr>, Horizontal Rule -->
    - 수평바, 구분자
    - 문단과 문단을 구분하는 역할
    - 단독 태그

    - 속성
        ~ size: 선의 두께
        ~ width: 선의 너비(픽셀: 절대크기 / %: 상대크기)
        ~ align: 수평 정렬
        ~ color: 선의 색상
        ~ noshade: 그림자 유무
        ~ titleL 풍선 도움말


<!-- <a>, anchor -->
    1. 링크, Link 역할

    - 속성
        ~ href: 클릭하면 이동하는 URL 지정
        ~ target:페이지 이동 방식
            - _self|_blank|_paren_top|프레임명
                ~ _self: 보고 있는 창에서 페이지 이동 > 내부 링크
                ~ _blank: 새 창에서 페이지 이동 > 외부 링크

    2. anchor 역할


<!-- <img>, image -->
    - 이미지
    - 단독 태그

    - 속성
        ~ src: 이미지 파일 URL > 필수 속성
        ~ alt: 대체 문자열 > 필수 속성
            - 이미지가 올바르게 표시되지 않았을 때 대신 보여주는 문자열
        ~ width
        ~ height
