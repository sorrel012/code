
<!-- <p>, paragraph -->
    - 문단 태그
    - 문단을 표현(=텍스트 집합)
    - 쌍태그(<p>문단의내용</p>)
    - 혼합용(텍스트 + 자식 태그)
    
    - 속성
        ~ align : 내용물의 수평 정렬
            - horizontal alignment
            - 열거형 > left|center|right|justify(양쪽정렬)
    

<!-- <hn>, heading -->
    - 제목 태그
    - n: 1~6
    - <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
    - 쌍태그
    - 혼합형
    - 숫자는 중요도를 나타낸다. > 보이는 형태는 중요하지 않음!
        ~ <h1>: 문서 내에서 가장 중요한 제목
        ~ <h6>: 문서 내에서 가장 덜 중요한 제목

    - 속성
        ~ align: 수평 정렬

<!-- <hr>, Horizontal Rule -->
    - 수평바, 구분자
    - 문단과 문단을 구분하는 역할
    - 단독 태그

    - 속성
        ~ size: 선의 두께
        ~ width: 선의 너비(픽셀: 절대크기 / %: 상대크기)
        ~ align: 수평 정렬
        ~ color: 선의 색상
        ~ noshade: 그림자 유무
        ~ titleL 풍선 도움말


<!-- <a>, anchor -->
    1. 링크, Link 역할

    - 속성
        ~ href: 클릭하면 이동하는 URL 지정
        ~ target:페이지 이동 방식
            - _self|_blank|_paren_top|프레임명
                ~ _self: 보고 있는 창에서 페이지 이동 > 내부 링크
                ~ _blank: 새 창에서 페이지 이동 > 외부 링크

    2. anchor 역할


<!-- <img>, image -->
    - 이미지
    - 단독 태그

    - 속성
        ~ src: 이미지 파일 URL > 필수 속성
        ~ alt: 대체 문자열 > 필수 속성
            - 이미지가 올바르게 표시되지 않았을 때 대신 보여주는 문자열
        ~ width
        ~ height


<!-- 인라인 태그 vs 블럭 태그 -->
    - 모든 태그는 위의 두 분류 중 하나에 속한다.
    - 렌더링(Rendering, 태그를 화면에 출력하는 작업) 규칙이 서로 다르다.

    - 인라인 태그, Inline target
        - 자신의 내용물과 앞뒤의 다른 태그의 내용물을 같은 라인에 출력한다.
        - 내용물의 크기가 태그의 영역이다.
        - 엔터를 치지 않으면 한 줄로 나타난다.
        - PCDATA(텍스트)는 인라인 태그로 취급한다.
        ~ 스타일 태그, <a>, <img>

    - 블럭 태그, Block Tag
        - 자신의 내용물과 앞뒤의 다른 태그의 내용물을 다른 라인에 출력한다.
        - 내용물의 크기와 상관없이 한 라인을 독점한다.(너비 100%) 높이는 내용물의 크기와 동일하다.
        - 엔터를 치지 않아도 줄바꿈이 된다.
        ~ <p>, <h1>....<h6> , blockquote, <ol>, <ul>, <dl>, <hr>;
