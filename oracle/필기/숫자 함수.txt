
-- 숫자 함수(= 수학 함수) , Numerical Function
	- Math.random()
	
	1. round()
		- 반올림 함수
		- number round(컬럼명) : 정수 반환
		- number round(컬럼명, 소수이하 자릿수) : 반올림하여 지정한 소수이하 자릿수까지의 실수 반환
		
	2. floor(), trunc()
		- 절삭 함수
		- 무조건 내림 함수
		- number floor(컬럼명) : 무조건 정수 반환
		- number trunc(컬럼명 [, 소수이하 자릿수]) : 정수 or 실수 반환
		
	3. ceil()
		- 무조건 올림 함수
		- number ceil(컬럼명) : 정수 반환
		
	4. mod()
		- 나머지 함수
		- number mod(피제수, 제수)
	
	5. power(), sqrt()
		- number power(밑, 지수)
		- number sqrt(제곱수) : 제곱근 반환
