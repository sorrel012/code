
--name is already used by an existing object
	- 이미 있는 이름을 또 사용
		> 이름을 변경하여 다시 시도
		
-- value larger than specified precision allowed for this column
	- 자료형의 범위를 벗어남 > overflow 발생
		> 자료형을 변경하거나, 값을 변경

-- not a single-group group function
	- 컬럼 리스트에 집계 함수와 일반 컬럼을 동시에 사용
		> 해결 불가.. 따로따로 가져와야 함. (어떻게든 할 수는 있지만 기본적으로는 x)

-- group function is not allowed here
	- where 절에 집계 함수를 사용
		> 해결 불가... where절은 집합값에 대한 접근이 불가능 (어떻게든 할 수는 있지만 기본적으로는 x)
		
--  unique constraint (HR.SYS_C007072) violated
	- unique 제약 조건을 위반
		> 위반한 값을 중복되지 않도록 수정

-- not enough values
	- 컬럼리스트의 컬럼 개수와 값리스트의 값 개수가 일치하지 않음.
		> 값리스트에서 빠뜨린 것이 있는지 확인

-- too many values
	- 컬럼리스트의 컬럼 개수와 값리스트의 값 개수가 일치하지 않음.
		> 컬럼리스트에서 빠뜨린 것이 있는지 확인하거나 값리스트에 추가로 들어간 것이 있는지 확인

-- column ambiguously defined
	- 동일한 컬럼명이 2개 이상 발견되었음.
		> 별칭을 다르게 설정해주기

-- integrity constraint () violated - parent key not found
	- 참조키를 설정했을 때, 부모 테이블에 해당 값이 존재하지 않음
		> 부모 테이블에 해당 값을 추가하기

-- table must be empty to add mandatory (NOT NULL) column
	- 테이블에 이미 데이터가 들어있을 때, not null 제약사항을 가지는 컬럼을 추가하려고 함.
		> 들어있는 데이터를 delete하고 생성하거나, 테이블을 drop하고 새로 create
