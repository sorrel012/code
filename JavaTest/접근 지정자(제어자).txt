// 접근 지정자(제어자), Access Modifier
	- 클래스 멤버 변수에 붙여서 사용한다.
	- 클래스 멤버 메소드에 붙여서 사용한다.
	- 클래스 자체에 붙여서 사용한다.
	- 멤버를 외부에 공개할지 말지 결정하는 역할 > 클래스가 경계선 역할
	   > 멤버 노출 수위를 조절하는 역할
	   > 외부에 멤버 공개 유무 조절하는 역할
	
	1. public
		- 외부에 멤버를 100% 공개
		
	2. private
		- 외부에 멤버를 100% 비공개
		
	3. protected
	
	4. default
