// 문자열 : 문자의 집합

	a. 길이
		~ int length()
		- 문자열을 구성하는 문자의 개수 > 글자 수
	
	b. 위치 추출
		~ char charAt(int index) 
		- 원하는 위치의 문자를 추출하는 메소드
		- Zero-based Index
		
		※ char 숫자 -> int 숫자 : char - 48; (0의 문자 코드값)
		
	c. 공백 제거
		~ String trim()
		- 문자열의 양끝(시작과 끝)에 존재하는 공백을 제거하는 메소드
		- 공백(Whitespace > 스페이스, 탭, 개행)
		
	d. 포함 여부 검색
		~  boolean contains(String) 
		- 특정 문자열이 포함되어 있는지 확인
		
	e. index 검색
		~ int indexOf(String s), int indexOf(char c), int indexOf(String s, int beginIndex),  int indexOf(char c, int beginIndex)
		~ int lastIndexOf(String s), int lastIndexOf(char c), int lastIndexOf(String s, int beginIndex),  int lastIndexOf(char c, int beginIndex)	
		- 없으면 -1 반환
		
	f. 대소문자 변환
		~ String toUpperCase(): 문자열을 모두 대문자로 변환
		~ String toLowerCase(): 문자열을 모두 소문자로 변환
	
	g. 대소문자 구분 없이 비교
		~ boolean equalsIgnoreCase(sTRING)
	
	h. 패턴 검색
		~ boolean startsWith(String): 특정 문자열로 시작하는지 확인
		~ boolean endsWith(String): 특정 문자열로 끝나는지 확인
		
	i. 문자열 추출
		~ String substring(int beginIndex, int endIndex) // endIndex는 포함되지 않음.
		~ String substring(int beginIndex)
		
	j. 문자열 치환
		~ String replace(String old, String new)
		- 문자열의 일부를 다른 문자열로 교체
		
	k. 문자열 분리
		~ String[] split(String deimiter) 
		- delimiter : 구분자 > 배열로 반환할 때 사라진다.
		
	l. 문자열 연결
		~ concat(String)
		- 두 문자열을 연결해 준다.
		- + 와 같은 역할
		

※ 1.메모리의 공간은 한 번 할당되면 더이상 늘리거나 줄일 수 없다.
※ 2.문자열은 불변(Immutable)이다. > 한 번 만들어진 문자열은 절대 수정할 수 없다.
※ 3.문자열은 참조형이다.
